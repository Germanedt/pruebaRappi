Esta proyecto fue desarrollada utilizando
    Angular
    Materialize

Para esto se utilizaron ciertas dependecias que estan descritas en el archivo package.json por lo cual antes de ejecutar el proyecto es necesario ejecutar en la terminal (CMD)
el siguiente comando (Teniendo en cuenta que debe estar posicionado dentro de la carpeta del proyecto)

1) npm install -g @angular/cli

2) npm install

Por ultimo para probar debe ejecutar el siguiente comando

3) ng serve
